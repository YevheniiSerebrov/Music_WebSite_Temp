First, the data are highly correlated, causing difficulty for the estimation process, and second, the unconstrained estimation produces estimates that imply an unstable system. For the second problem we will use CML to constrain the eigenvalues of to be less than one in absolute value. The program for estimating the coefficients in this model, cklein.prg,is written in the GAUSS programming language and uses Aptech System's Constrained Maximum Likelihood (CML) applications module.
